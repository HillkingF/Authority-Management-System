index.js中:

export default new Vuex.Store({
  state: {
    //这里放全局参数
  },
  mutations: {
    //这里是set方法: 可对全局参数进行操作赋值
  },
  getters: {
    //get方法: 对全局参数操作
  },
  actions: {
  },
  modules: {
    // 这里是我自己理解的是为了给全局变量分组，
    // 所以需要写提前声明其他store文件，然后引入这里
  }
})

========================================================
为了区分不同的模块组件之间的参数,创建单独的目录及js文件对不同组件的参数进行读写
如: modules 目录中的menus.js 是对左侧导航栏中的参数/状态进行编辑的
